2016年暑假总结，不算总结的总结

发表于 2016-08-22   |     |   阅读次数

转眼间2016年的暑假已经接近尾声了，在这个时候很有必要写篇文章回顾一下整个暑假的历程，看看都干了些什么。

在娱乐方面，我把美剧生活大爆炸全部看完了，还有闪电侠第二季也看完了，也看了电影 控方证人 疯狂的石头 低俗小说 教父三部曲 心慌方三部曲 和 异形。我一直觉得这是很神奇的事情，我们生活的这个时代是一个很好的时代，即使是十几年甚至几十年前的电影，我们现在还能看见，还能欣赏，这得益于网络的发展以及各方面的因素才能造就这样的结果。尤其是生活大爆炸，拍了十几年的美剧，我花了不到一个月的时间就把它全部看完了，这感觉真TM爽，不用追剧，一看就把它全部看完。

看书方面的话，把DOOM启世录这本书给看完了，当然也没少看各种各样的文章，也看了好几期的知乎周刊。

之前保存了王登科的博客的书签，一直没看，趁着放假的时候，零零星星发了两天的时间把他的博文全部看完了，这厮从大学开始写博客，一直坚持，现在他都大学毕业了，他的博客记录着他的成长经历，我觉得挺好，我也把他的友情链接也全打开了，发现有的已经打不开了，可能是他们不能坚持着，所以导致博客关闭了，我无从得知。从他的博客，我发现了王子亭的博客，这家伙更加厉害，从小学就开始写网络日记了，一直坚持到现在，他在高中退学，进入互联网行业工作，目前在 LeanCloud 任 Node.js 服务器端开发工程师。他虽然是1995年出生的，可他的人生经历真的很丰富，看他博客，文字从稚嫩到成熟，一步步记录着自己的生活，也写了很多跟技术相关的文章。我十分敬佩这样的人。也希望能成为这样的人。优秀的人的朋友也不会差到那里去，所以我也拜访他的友情链接，发现他们都是很厉害的人。有的还在读初中就已经懂得前端，而且还玩得十分溜了。有的高考考了三次，只为考上自己理想中的大学，他有明确的目标，对自己提出类似于五年计划的计划，我也看见了，他自己提出来自己都慢慢实现了，可能对自己严格要求的人未来都不会很差吧，他目标清晰，他的未来他自己可以预见。王子亭这样的朋友或者说是粉丝还有很多很多，他甚至开了自己的粉丝群，可能大家都喜欢优秀的人吧。群里很多人。

还有就是晨旭，他也开了博客，我知道他是因为在网站逛的时候发现有人写了博客专门介绍了他，这小子也是玩的挺溜的，做了一个虚假磁力链接，别人做短网址，他反其道而行之，做了一个长网址，从这里看出，他是有趣的人。所以我就追到了他的博客，发现果然没错，他经常发一些有趣的东西，从他的一篇文章中发现了太空小孩，发现他也是挺有趣的人，偶尔发下技术文，还是饭否的忠实用户，也在晨旭的博客发现了很多友情链接了，友情链接之后还有更多的友情链接。总之到现在我还没探索完。

我就在思考，建一个博客是一件多么有趣的事情，我是不是也可以建一个属于自己博客呢，说干就干，正好腾讯有一个云+校园计划，于是实名认证，学生认证之后，优惠券就到账，域名就到手了，接下来就是选择什么博客程序的问题，其实这个问题还是挺纠结的，博客程序这么多，选哪个好呢，纠结来纠结去，最后选择了Hexo，静态博客，只需专注内容，不涉及数据库什么的，而且很漂亮，所以就它了。一开始我把它托管在GitHub上，前几天晨旭来访我的博客的时候发现我这网站加载速度贼慢，所以我把它同时部署在GitHub和Coding上，发现速度大大加快了。所以博客就这么建好了，以后就是专注于记录写文章的事了。还有就是多串串门，找点邻居才行。

要说暑假最大的收获还是学会用小飞机到墙外的世界看看，虽然之前也用过free gate出去看看过，也用过lantern看过，但是当free gate和lantern遇到小飞机，那简直是小巫见大巫了，虽然free gate和lantern很傻瓜式，打开就能用了，但是他们的速度什么还是很令人堪忧的，而且lantern最近还推出了收费版，免费版限定800M流量，用完了就没了，逛论坛的时候发现小飞机，其实他也不是很复杂，只要有账号密码，配置一下就可以使用了，所以总的来说还是十分的方便，而且我在论坛发现了很多人把自己的账号分享出来给大家用，这是一件很棒的事情。目前来说，我用小飞机可以访问到YouTube，Twitter，Google，，还有一些我们平时访问不了的网站，尤其是Google，Google真的比百度好用不知道多少倍，Gmail也很好，所以我也注册了一个Gmail账号，还有Wikipedia，可以上去查一些东西，还有就是Instagram，偶尔上去看两张美图也是一件很不错的事情。目前还在不断探索之中。有小飞机，以后飞到墙外就是一件很容易的事情了。

要说暑假有什么遗憾的话，还是有的，本来计划着在暑假开始学习Web前端的，可是由于拖延症，一直都没有开始，所以这个计划又要推迟了；还有就是说要跑步的，可是没坚持几天就不再跑了，这是一件多么遗憾的事呀。

过几天就要去上学了，所以就这样吧。

#总结
